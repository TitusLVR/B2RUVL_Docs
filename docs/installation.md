# Installation & Configuration

## Installation

1.  Download the latest release of the **B2RUVL** addon as a `.zip` file.
2.  Open Blender.
3.  Go to **Edit** > **Preferences** > **Add-ons**.
4.  Click **Install...** and select the downloaded `.zip` file.
5.  Enable the addon by checking the box next to **UV: B2RUVL**.

## Configuration

Before using the addon, you must configure the paths to your external applications.

1.  In the Addon Preferences (expand the B2RUVL entry in the Add-ons list):
2.  **RizomUV Setup**:
    *   Check **Use RizomUV**.
    *   **Executable Path**: Browse to your RizomUV executable (e.g., `C:\Program Files\Rizom Lab\RizomUV 2025.0\rizomuv.exe`).
    *   **Scripts Path**: This should default to the addon's internal scripts folder. Unless you have custom scripts, leave this as is.
3.  **Headus UVLayout Setup**:
    *   Check **Use UVLayout**.
    *   **Executable Path**: Browse to your Headus UVLayout executable.

## Usage Locations

Once installed and enabled, the B2RUVL panel can be found in:
*   **3D Viewport** > **Sidebar (N-Panel)** > **B2RUVL** (or the category name you set in preferences).
